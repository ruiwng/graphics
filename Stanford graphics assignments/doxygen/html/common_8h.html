Create a std::unique_ptr much like std::make_shared.

To avoid too many compilation problems I stuck with using C++11 where the committee forgot to include std::make_unique (it's there in C++14). This is a convenience function. Curious about Modern C++? The book Effective Modern C++ by Scott Meyes is a great book if you are already familiar with C++.
